# Ant Date 日期选择器组件

基于 Ant Design Vue 的日期选择器组件，支持多种日期类型、时间选择、范围选择等功能，使用 dayjs 进行日期处理。

## 基础用法

```vue
<template>
  <ant-date v-model="dateValue"/>
</template>
<script> export default {
  data() {
    return {dateValue: '2024-01-01'}
  }
} </script>
```

## 📅 日期格式说明

### 常用格式标记

| 标记 | 说明 | 示例 |
|------|------|------|
| `YYYY` | 四位年份 | 2024 |
| `YY` | 两位年份 | 24 |
| `MM` | 月份（补零） | 01-12 |
| `M` | 月份 | 1-12 |
| `DD` | 日期（补零） | 01-31 |
| `D` | 日期 | 1-31 |
| `HH` | 小时24制（补零） | 00-23 |
| `H` | 小时24制 | 0-23 |
| `hh` | 小时12制（补零） | 01-12 |
| `h` | 小时12制 | 1-12 |
| `mm` | 分钟（补零） | 00-59 |
| `m` | 分钟 | 0-59 |
| `ss` | 秒数（补零） | 00-59 |
| `s` | 秒数 | 0-59 |
| `SSS` | 毫秒 | 000-999 |
| `A` | AM/PM | AM, PM |
| `a` | am/pm | am, pm |
| `d` | 星期几 | 0-6 |
| `ddd` | 星期缩写 | Mon-Sun |
| `dddd` | 星期全称 | Monday-Sunday |
| `wo` | 第几周 | 1w, 2w... |

## 📅 pickerType 类型说明

`pickerType` 属性用于控制日期选择器面板的展示维度。当 `type` 为 `ant-date`（单选）或 `ant-daterange`（范围）时，通过配置 `pickerType` 可以实现年、季、月、周、日等不同粒度的选择。

| pickerType 值 | 说明 | 默认格式参考 (format) | 适用 type | 返回值示例 (valueFormat="YYYY-MM-DD") |
| :--- | :--- | :--- | :--- | :--- |
| **`date`** | 选择天（默认值） | `YYYY-MM-DD` | ant-date, ant-daterange | `"2024-01-01"` |
| **`week`** | 选择周 | `YYYY-wo` | ant-date, ant-daterange | `"2024-1wo"` (第1周) |
| **`month`** | 选择月 | `YYYY-MM` | ant-date, ant-daterange | `"2024-01"` |
| **`quarter`** | 选择季度 | `YYYY-QQ` | ant-date, ant-daterange | `"2024-Q1"` |
| **`year`** | 选择年 | `YYYY` | ant-date, ant-daterange | `"2024"` |

## 返回值说明

### 无 valueFormat 时

- **单个日期**：返回时间戳（Number）
- **日期范围**：返回时间戳数组（Number[]）

```javascript
// 单个日期
dateValue = 1704067200000 // 2024-01-01 的时间戳
// 日期范围
dateRange = [1704067200000, 1706659200000]
```

### 有 valueFormat 时

- **单个日期**：返回格式化字符串
- **日期范围**：返回格式化字符串数组

```javascript
// value-format="YYYY-MM-DD"
dateValue = "2024-01-01"
dateRange = ["2024-01-01", "2024-01-31"]
// value-format="YYYY-MM-DD HH:mm:ss"
dateTimeValue = "2024-01-01 12:00:00"
```

## 注意事项

1. **dayjs 依赖**：组件内部使用 dayjs 进行日期处理，已自动引入必要的插件（weekday、localeData、customParseFormat）

2. **国际化**：默认

[//]: # (3. 使用中文 locale（zh-cn）)

3. **valueFormat 的重要性**：
   - 不设置时返回时间戳，适合后端存储
   - 设置后返回格式化字符串，适合直接展示
   - 建议始终设置 valueFormat 以保证数据格式一致

4. **范围选择器的值**：
   - 空值时为数组 `[]`
   - 有值时为 `[startDate, endDate]`

5. **日期格式参考**：
   - `YYYY` - 四位年份
   - `MM` - 月份（01-12）
   - `DD` - 日期（01-31）
   - `HH` - 小时（00-23）
   - `mm` - 分钟（00-59）
   - `ss` - 秒（00-59）
   - `WW` - 周数

6. **性能优化**：组件内部做了值变化检测，避免不必要的更新循环

7. **popperAppendToBody**：默认为 true，在模态框等特殊场景中可能需要设置为 false

8. **disabledDate 函数**：接收的参数是 JavaScript Date 对象，不是 dayjs 对象

## 常见问题

### Q1: 如何获取当前选中的 dayjs 对象？

A: 在 change 事件的第二个参数中可以获取：

```javascript
handleChange(value, date)
{
  console.log('dayjs对象:', date)
  console.log('格式化:', date.format('YYYY-MM-DD'))
}
```

### Q2: 范围选择器如何设置初始值？

A: 传递数组格式：

```javascript
data()
{
  return {dateRange: ['2024-01-01', '2024-01-31']}
}
```

### Q3: 如何禁用周末？

```javascript
disabledDate(current)
{
  return current && current.getDay() === 0 || current.getDay() === 6
}
```

### Q4: 如何只显示未来日期？

```javascript
disabledDate(current)
{
  return current && current < new Date().setHours(0, 0, 0, 0)
}
```
