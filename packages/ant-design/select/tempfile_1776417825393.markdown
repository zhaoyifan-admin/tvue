# TVUE Ant Design Select 组件文档

## 组件说明

基于 Ant Design Vue 的 Select 选择器组件，提供了丰富的功能扩展，包括多选、搜索、远程加载、动态添加选项等特性。

## Props 参数说明

| 参数                   | 说明       | 类型                                   | 默认值   | 可选值                      |

## Slots 插槽

### default

```vue

<tvue-ant-select v-model="value" :dic-data="options">
  <template #default="{ item, label, value }">
    <div>
      <span>{{ item[label] }}</span>
      <span v-if="item.desc">{{ item.desc }}</span>
    </div>
  </template>
</tvue-ant-select>
```
## 注意事项

1. **动态添加选项**：使用 `allowAdd` 时，建议同时设置 `filterable: true` 以获得更好的用户体验

2. **远程搜索**：使用 `remote: true` 时，需要配置 `dicUrl` 或在外部监听搜索事件

3. **虚拟滚动**：当数据量超过 1000 条时，建议启用 `virtualize: true` 以提升性能

4. **全选功能**：`all: true` 仅在 `multiple: true` 时生效

5. **值类型**：注意 `isJson`、`isString`、`isNumber` 等属性会影响最终输出的值格式

6. **自定义渲染**：使用 `dropdownRender` 时，必须手动渲染 `menu` 参数才能显示选项列表

7. **消息提示**：组件内部使用 `window.message` 进行消息提示，需要在应用中配置全局 message 对象

## 常见问题

### Q: 如何清空选择器？
A: 将绑定值设置为 `undefined`、`null` 或空字符串即可。

### Q: 多选时如何限制最大选择数量？
A: 设置 `limit` 属性，例如 `limit: 3` 表示最多选择 3 项。

### Q: 如何自定义选项的显示内容？
A: 使用默认插槽，通过 slot-scope 获取 item、label、value 等数据进行自定义渲染。

### Q: 远程搜索如何实现？
A: 设置 `remote: true` 和 `filterable: true`，并配置 `dicUrl`，组件会自动在用户输入时发起请求。

### Q: 添加的选项如何持久化？
A: 通过 `onAddOption` 回调函数，在回调中将新选项保存到服务器或本地存储。
