### 快速上手

```bash
# 安装
yarn add  @zhaoyifannan/tvue -S # 或者：npm i @zhaoyifannan/tvue -S

# 引入
import Tvue from '@zhaoyifannan/tvue';
import '@zhaoyifannan/tvue/lib/index.css';
Vue.use(Tvue);

```
